SetOutputConsumer(ActionIEnumerableTupleString, Object)
Designates an output consumer action. This overload of the SetOutputConsumer method is intended for use in case of record-centric output to a single target and no need to access the global cache. To use this overload, the OutputDataKind setting must be either XML or JSON. See the note below for differences between these two kinds.
SetOutputConsumer(ActionString)
Designates an output consumer action. This overload of the SetOutputConsumer method is intended for use in case of textual output (such as Delimited, Flat, etc.) to a single target and no need to access the global cache.
